## Archive Instructions

This document receives content from VNPC_Status.md to keep the main status doc manageable.

### Version History
- VNPC_Status.md keeps only the **9 most recent** version entries
- When a session adds a new version, move the oldest entry here
- Add to the TOP of the Version History table (newest archived first)

### Session Summaries
- VNPC_Status.md keeps only the **Last Session + 5 most recent Previous** summaries
- Archive older session summaries here when they roll off

### Completed Milestones
- Move milestone sections here when marked **COMPLETE** in VNPC_Status.md
- Replace with a one-line reference in the active doc
- Include the full milestone text as it existed when archived

**Do NOT archive (keep in active doc):**
- Active/in-progress milestones
- Coding standards and AI rules
- Architecture and assembly definitions
- Reference documents table
- Session close checklist
