## Archive Instructions

This document receives content from VNPC_CodeReference.md when:
- A script is **deleted** -- move its full entry here with a note on why
- A script undergoes **major refactor** -- archive the old API, update the active doc with the new API
- An assembly is **removed or merged** -- archive the old assembly entry

**Do NOT archive:**
- Minor API changes (just update the active doc)
- Bug fixes that don't change the public API
